A vanilla note-taking app

Edit Remove
  • Click Create button to write note
  • Hover on folder to see buttons
  • Double click folder to toggle
  • Hover on note title in note list to get delete button
  • Note title are auto generated, I don't want to write it manually
  • You can also create top-level folder via select list
  • In select list, you can choose what folder the note will save to

This is merely a vanilla mvc demo with zero dependency, which uses your localStorage to save data, the space is limited to 10MB on desktops.
